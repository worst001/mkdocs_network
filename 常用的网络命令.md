SSH: SSH（Secure Shell）是一种用于远程登录和安全传输数据的网络协议。

Telnet：Telnet 是一个用于远程登录到远程主机的协议和工具。它允许您通过命令行界面与远程计算机进行通信。

Ping：Ping 工具用于测试网络连接的可达性和延迟。它发送一个 ICMP 回显请求消息到目标主机，并接收回复以测量往返时间。

Traceroute：Traceroute 工具用于确定数据包从源主机到目标主机的路径。它显示了数据包经过的每个中间节点（路由器），以及每个节点的延迟。

nslookup：nslookup 是一个用于查询 DNS（域名系统）记录的实用工具。它可以用来查找主机名的 IP 地址，或反向查找 IP 地址的主机名。

Netcat：Netcat 是一个网络工具，可以在不同的计算机之间建立 TCP 或 UDP 连接，并进行数据传输。它可以用作端口扫描器、简单的网络代理、文件传输工具等。

Wireshark：Wireshark 是一个流行的网络协议分析工具，可以捕获和分析网络数据包。它可以帮助您检查和解决网络故障，并深入了解网络通信的细节。

Nmap：Nmap 是一个强大的端口扫描工具，可以用于发现网络上的主机和开放的端口。它还提供了一些高级功能，如操作系统识别、服务版本检测等。

Curl：Curl 是一个命令行工具，用于发送 HTTP 请求并获取响应。它支持多种协议，包括 HTTP、HTTPS、FTP 等，可用于测试网站的可访问性和性能。
